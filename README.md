# RepCounter

Minimalistische workouttracker voor Push, Pull en Legs. Logt iedere set met datum en timestamp, toont e1RM-progressie en geeft veilig PR-advies.

## Starten

Dubbelklik op `Start RepCounter.bat` of gebruik `npm install` en `npm run dev`.

## Opslag

Versie 0.1.0 gebruikt browser `localStorage`. Daardoor werkt de app zonder account en blijven gegevens op hetzelfde apparaat beschikbaar.

## Turbo Repo Hub

De repository bevat `turbo-project.json`, `VERSION`, een healthcheck en een Windows-launcher voor universele herkenning.
